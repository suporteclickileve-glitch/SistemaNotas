# 📊 Sistema de Gestão de Notas

Aplicação desenvolvida em Java para automatizar o controle acadêmico, o cálculo de médias e o acompanhamento do desempenho de estudantes. Ideal para gerenciar turmas, disciplinas e notas de forma simples e eficiente.

## 🚀 Funcionalidades

- **Cadastro de Alunos**: Registro com nome, matrícula e turma.
- **Gestão de Disciplinas**: Inclusão das matérias ofertadas.
- **Lançamento de Notas**: Entrada de notas parciais (ex: Prova 1, Prova 2 e Trabalhos).
- **Cálculo Automático**: Média aritmética ponderada ou simples definida por disciplina.
- **Verificação de Status**: Classificação automática como *Aprovado*, *Reprovado* ou *Em Exame*.
- **Relatórios**: Exibição do boletim individual do aluno ou geral da turma.

## 🛠️ Tecnologias Utilizadas

- **Linguagem**: Java (Versão 17 ou superior)
- **Paradigma**: Programação Orientada a Objetos (POO)
- **IDE**: IntelliJ IDEA ou Eclipse
- **Gerenciador de Dependências** (Opcional): Maven ou Gradle

## ⚙️ Como Executar o Projeto

1. Clone o repositório para a sua máquina:
   ```bash
   git clone https://github.com
   ```
2. Abra o projeto na sua IDE de preferência.
3. Certifique-se de configurar o **JDK 17+** no seu ambiente.
4. Execute a classe principal `Main` para iniciar o sistema.

## 📋 Exemplo de Uso

O sistema é executado via terminal. O fluxo básico consiste em:
1. Adicionar um novo aluno.
2. Selecionar o aluno e adicionar as notas da disciplina.
3. Visualizar o boletim gerado com a média final e situação.
